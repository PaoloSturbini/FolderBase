# FolderBase Roadmap

This roadmap communicates direction rather than fixed delivery commitments.

## Current priorities

- Improve onboarding and first-run guidance.
- Expand automated tests for database and filesystem-sensitive operations.
- Improve English documentation and screenshots.
- Strengthen semantic search and document-chat reliability.
- Improve accessibility and keyboard navigation.
- Validate compatibility across supported macOS versions and Apple Silicon configurations.

## Distribution

- Add repeatable release checks and SHA-256 checksums.
- Evaluate Developer ID signing and Apple notarization.
- Improve update and migration documentation.

## Community

- Curate `good first issue` and `help wanted` tasks.
- Add contribution-oriented architecture notes.
- Collect structured feedback from users managing large document libraries.

Feature proposals are welcome through GitHub Issues. Inclusion here does not guarantee implementation or timing.
